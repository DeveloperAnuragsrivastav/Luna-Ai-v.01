## Installation ğŸ”§

1. Clone the repository:
```bash
git clone https://github.com/DeveloperAnuragsrivastav/Luna-Ai.git
cd Luna-Ai
````

2. Install dependencies for frontend and backend:

```bash
npm install
```

3. Run the server:

```bash
node server.js
```

4. Run the React frontend:

```bash
npm run dev
```

5. Open your browser at:

```
http://localhost:5173
```
